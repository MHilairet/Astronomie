# Système d'alignement polaire automatique ouvert / Open Automatic Polar Alignment System
> Un système motorisé à deux axes (azimut et altitude) pour l'alignement polaire automatique avec TPPA dans NINA ou l'alignement polaire manuel avec un exécutable Windows d'AVALON 

> A two‑axis (Azimuth & Altitude) motorized system for automatic polar‑alignment with TPPA in NINA or 
manual polar‑alignment with an Windows executable file from AVALON.

Le code tourne sur une carte Arduino UNO et émule le protocol Avalon de l'UPAS. Dans la première version, le code a été testé pour des drivers de puissance A4988 et TMC2130.

The code runs on an Arduino UNO board and emulates the UPAS Avalon protocol. In the first version, the code was tested with A4988 and TMC2130 power drivers.
